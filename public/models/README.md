# 3D Model Assets

Drop your GLB file here as `avatar.glb`.

See `/docs/08-3d-avatar-setup.md` for step-by-step instructions on getting a free 3D avatar.

## Quick start

The fastest option: download `CesiumMan.glb` from https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/CesiumMan/glTF-Binary, rename to `avatar.glb`, and drop it in this folder.

## Files used by the app

| File | Where it's used |
|---|---|
| `avatar.glb` | Default — shown in both Twin widgets |

Optional overrides can be passed via the `avatarUrl` prop in code.
