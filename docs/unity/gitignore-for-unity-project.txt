# ═════════════════════════════════════════════════════════════════════
# .gitignore for the purex-unity-avatar Unity project
# ─────────────────────────────────────────────────────────────────────
# When you create the Unity project (Step 3 of docs/unity-as-library.md,
# Phase 1), copy this file to the project root and rename to `.gitignore`:
#
#   cp purex-phase1/docs/unity/gitignore-for-unity-project.txt \
#      purex-unity-avatar/.gitignore
#
# This is the standard Unity .gitignore plus a few project-specific
# additions for our pipeline (FBX cache, Mixamo downloads, MakeHuman
# scratch files we don't commit).
# ═════════════════════════════════════════════════════════════════════
